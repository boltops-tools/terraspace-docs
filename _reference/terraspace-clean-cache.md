## Usage

    terraspace clean cache

## Description

Removes cache dirs.

## Example

    $ terraspace clean cache
    Will remove these folders and all their files:

        .terraspace-cache
        /tmp/terraspace

    Are you sure? (y/N)
    Removed .terraspace-cache
    Removed /tmp/terraspace
    $


## Options

```
-y, [--yes], [--no-yes]  # bypass are you sure prompt
```
