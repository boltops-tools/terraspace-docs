## Usage

    terraspace clean all

## Description

Runs all clean operations.

## Examples

    $ terraspace clean all
    Will remove Terraspace cache and logs.
    Are you sure? (y/N) y
    Removed .terraspace-cache
    Removed /tmp/terraspace
    Removing all files in log/
    Logs removed
    $


## Options

```
-y, [--yes], [--no-yes]  # bypass are you sure prompt
```
