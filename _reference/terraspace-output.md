## Usage

    terraspace output STACK

## Description

Run output.

## Example

    $ terraspace output demo
    => terraform output
    bucket_name = bucket-fond-sheep
    $


## Options

```
i, [--instance=INSTANCE]         # Instance of stack
o, [--out=OUT]                   # Write the output to path
    [--verbose], [--no-verbose]  
    [--noop], [--no-noop]        
```
